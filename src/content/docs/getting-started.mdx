# Getting Started

Welcome to **Gestalt** — your desktop-style docs and scripts hub.

## Installation

```bash
bun create astro@latest gestalt
cd gestalt
bun install
```

## Project Structure

```
gestalt/
├── src/
│   ├── components/    # Vue + Astro components
│   ├── content/       # MDX docs and scripts
│   ├── pages/         # Astro pages
│   └── styles/        # Global CSS
├── public/            # Static assets
└── astro.config.mjs   # Astro configuration
```

## Features

- Draggable desktop icons with persistent positions
- App windows with minimize, maximize, and close
- Hyprland-inspired Catppuccin theme
- MDX-powered documentation

## Quick Tips

> Use the right-click context menu on the desktop to clear cached icon positions.

| Shortcut | Action |
|----------|--------|
| Click icon | Open window |
| Drag icon | Move icon |
| Drag title bar | Move window |
| Bottom-right corner | Resize window |
